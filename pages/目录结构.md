# 目录结构

---

# 简单组件

对于简单的组件，可以将组件的样式和逻辑写在同一个文件中。

```
components
├── ComponentAName
│   ├── index.module.css
│   ├── index.tsx
├── ComponentBName
│   ├── index.module.css
│   ├── index.tsx
```

---

# 复杂组件

对于较复杂的组件，应该将组件拆分成多个文件，每个文件对应一个子组件，最后在 index.tsx 中导出。


```
components
├── ComponentName
│   ├── ComponentOne.tsx
│   ├── ComponentOne.module.css
│   ├── ComponentTwo.tsx
│   ├── ComponentTwo.module.css
│   ├── index.tsx
```
